# MOC (Maps of Contents) 一覧

このディレクトリには、各視点（Perspective）別のMOCノートが格納されています。

## MOCとは？

MOC（Map of Contents）は、特定の視点やテーマで論文を整理するためのインデックスノートです。
Dataviewクエリを使用して、動的に論文リストを生成します。

## 自動生成

新しい論文を追加すると、該当するMOCノートが自動的に作成・更新されます。
